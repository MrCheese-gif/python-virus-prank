### What it does:

The programs, when running, will say random phrases in a robotic voice at a set interval. The file `virus.py` is meant for MacOS while `winVirus.py` was made for Windows. 

## Requirements:

- Python

### Install Requirements:

Install python from the [official website](https://www.python.org).

Depending on your Operating System, (`virus.py` for MacOS and `winVirus.py` for Windows)

## Run code

Open Terminal (on Mac or Windows) or Command Prompt (on Windows), navigate to the directory containing your code using `cd`, and then type:

```bash
python3 virus.py
```

for MacOS, and

```powershell
python virus.py
```

for Windows. Immediately, the program will activate. Ensure your volume is not set to 0 to hear the voice.

> **NOTE:** This program does not work on Linux

## End Code

To end the program, simply press `^C` or `ctrl + C` or `C-c`(all three refer to the same buttons) into the Terminal window. A message will then pop up in the terminal to confirm that the program has ended.
